Road Accident Analysis Dashboard (2021 - 2022)
📌 Project Overview
This project involves a comprehensive analysis of road accident data for the years 2021 and 2022. The primary goal was to identify patterns in casualties based on vehicle types, road conditions, and environmental factors to provide actionable insights for road safety.
📊 Key Insights & Features
Total Casualties: Analyzed a total of 417,883 casualties.
Vehicle Breakdown: Identified that Cars account for the highest number of casualties (333,485).
Trend Analysis: Compared accident trends between 2021 and 2022 to visualize year-over-year growth/decline.
Environmental Factors: Casualties were categorized by road surface (Dry/Wet), light conditions (Daylight/Dark), and location (Urban/Rural).
Interactive Dashboard: Used Slicers to allow users to filter data dynamically by date and area.
🛠️ Technical Skills Used
Data Cleaning: Processed raw data to handle missing values and ensure consistency.
Data Visualization: Created an interactive dashboard using Pivot Charts and customized themes.
Pivot Tables: Summarized large datasets to extract meaningful KPIs (Fatal, Serious, and Slight casualties).
Excel Formulas: Utilized advanced formulas for data categorization and trend calculations.
